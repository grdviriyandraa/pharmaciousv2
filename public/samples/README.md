# Foto contoh untuk tombol "Coba contoh"

Taruh 3–4 foto kulit manggis asli di folder ini dengan nama **persis** berikut
(halaman Inspeksi otomatis menampilkannya sebagai thumbnail; yang belum ada disembunyikan):

```
manggis-1.jpg
manggis-2.jpg
manggis-3.jpg
manggis-4.jpg
```

Tips: pakai foto latar polos + buah jelas (seperti 2 foto studio yang kamu kirim) supaya
demo paling meyakinkan. Format .jpg/.jpeg. Ukuran bebas (nanti di-resize 224×224 saat inferensi).

Empat foto yang kamu paste ke chat belum bisa aku ambil otomatis (itu tempelan gambar, bukan
file). Simpan saja keempatnya ke folder ini dengan nama di atas — begitu ada, thumbnail-nya
langsung muncul tanpa perubahan kode.
